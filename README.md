# Image Resizing API

</br>

#### ***Desenvolvido em***

- Linguagem: Go
- Versão: 1.24+

</br>
<hr>

#### ***Descrição***

O objetivo desta API é: 
- Receber uma imagem com um determinado tamanho, por exemplo: 1200 x 1480
- Processar a imagem
- Redefinir o tamanho da imagem para outro e retornar para o cliente
- **[Bônus]**: Também podemos mudar o formato da imagem para: .jpeg, .png, .ico, .webapp ou .svg
